## Each line in this file is an entry in the dictionary. An entry
## consists of several fields, each delimited by ‘|’. No field may
## contain a ‘|’ character. The fields are as follows:
##
##     1. The word in the UF orthography.
##     2. The part of speech and other initial annotations.
##     3. The etymology of the word. If this field contains no spaces, it
##        is assumed to be a PF word.
##     4. The English translation of the word.
##     5. (optional) Grammatical forms of the word.
##
## To break an entry across multiple lines, end each line with a ‘\’
## character. There MUST NOT be any characters between the ‘\’ and the
## end of the line. The ‘\’ character will be removed and the next line
## will be concatenated to the current line.
##
## Furthermore, it is possible to create references to other entries by
## adding a line that contains NO ‘|’ characters and is of the form
##
##     entries > reference
##
## where ‘entries’ is a comma-separated list of words for which to create
## references, and ‘reference’ is the word to which the references should
## point.
